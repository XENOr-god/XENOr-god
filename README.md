# XENØr

XENØr is a protocol and systems research stack. Public work is organized around deterministic execution, simulation-led validation, and verifiable incentive design, with each surface kept explicit and inspectable.

## Current focus

- xenor-core — deterministic execution core for protocol logic and incentive routing
- xenor-sim — simulation and scenario-validation environment for protocol behavior
- xenor-site — canonical public surface for architecture, repository boundaries, and public references

## Additional systems work

- xenor-engine — deterministic fixed-timestep simulation core for replay-oriented validation

## Archived / research record

- xenor-sale — archived Solana sale prototype kept as historical research context
