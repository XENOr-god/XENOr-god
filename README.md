# XENØr

XENØr is a protocol and systems research stack. Public work is organized around deterministic execution, simulation-led validation, and verifiable incentive design, with each surface kept explicit and inspectable.

## Current Focus

- [`xenor-core`](https://github.com/XENOr-god/xenor-core) — deterministic execution core for protocol logic and incentive routing
- [`xenor-sim`](https://github.com/XENOr-god/xenor-sim) — simulation and scenario-validation environment for protocol behavior
- [`xenor-site`](https://github.com/XENOr-god/xenor-site) — canonical public surface for architecture, repository boundaries, and public references

## Additional Systems Work

- [`xenor-engine`](https://github.com/XENOr-god/xenor-engine) — deterministic fixed-timestep simulation core for replay-oriented validation
